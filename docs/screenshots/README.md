# Screenshot Assets

Use this directory for documentation screenshots.

- `popup.png`: popup UI screenshot used in root README files.
- `sidebar.png`: sidebar UI screenshot used in root README files.
- `examples/`: screenshots for pages in `examples/`.

Recommended naming for `examples/` screenshots:

- `home.png`
- `todolist.png`
- `form.png`
- `textarea.png`
- `table.png`
- `spreadsheet.png`
- `presentation.png`
- `chat-with-your-data.png`
- `sdk.png`
